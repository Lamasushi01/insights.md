# 📊 Parent Media Insights

## 🎯 Key Takeaways

### 1. WhatsApp is the Primary Platform
Parents use WhatsApp daily → Best for retention & communication.

### 2. Instagram Drives Discovery
Parents discover schools via Instagram → Reels strategy is critical.

### 3. Late Night is Prime Time
Peak usage after 8 PM → Campaigns should focus on this window.

### 4. Content > Advertising
Parents prefer educational & parenting content → Value-first strategy required.

### 5. Trust = Other Parents
Recommendations & reviews influence decisions → Testimonials are key.

### 6. Outdoor Needs Digital Support
Hoardings alone are not effective → Must be backed by digital campaigns.

### 7. Commute = Passive Attention
Parents don’t engage deeply → Use simple, visual messaging.

---

## 🚀 Strategic Direction

- Instagram for awareness
- Google for intent capture
- WhatsApp for nurturing
- Testimonials for trust-building
